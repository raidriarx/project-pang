# Cropping job — how to do it (no git needed)

Thanks for helping! The task: **crop each photo so only the Buddha statue remains**, and send the cropped photos back. Everything happens in your browser + any photo app.

## 1. Download

Click **`to-crop-batch1.zip`** in this folder → press the **⬇ Download raw file** button → unzip it.

Inside are 5 folders (one per pose): `marawichai/`, `samathi/`, `nakprok/`, `prathanphon/`, `saiyat/`. Together: **111 photos**.

## 2. Crop (any app works — Preview, Photos, Paint, phone gallery…)

For every photo, crop to the **one statue** the folder's pose describes:

- ✅ Include the **whole statue**: the serpent/naga hood **above** the head (nakprok!), and the base/pedestal it sits on. The coiled snake under a nakprok Buddha **is part of the statue**.
- ✅ When unsure, crop a little **too wide** rather than too tight.
- ✅ If several statues are in the photo, crop the **biggest/front one that matches the folder's pose**.
- ❌ Don't rotate, resize, filter, or edit anything — just crop.
- ❗ **Keep the exact same filename** (e.g. `00616.jpg` stays `00616.jpg`) and keep it inside the same-named pose folder. The numbers are how we match everything back.

Pose reminders: marawichai = right hand on knee pointing down · samathi = both hands stacked in the lap · nakprok = snake hood over the head · prathanphon = right hand open palm-up on the knee · saiyat = lying down.

If a photo seems impossible (statue barely visible, wrong pose, etc.) — skip it, and note the filename in a `skipped.txt`.

## 3. Send back

1. Zip your cropped folders → name it **`cropped-batch1.zip`**.
2. On this GitHub page, open the **`cropping`** folder → **Add file → Upload files** → drag your zip in → press **Commit changes**.

That's it. New batches will appear here as `to-crop-batch2.zip`, etc., as more photos get labeled.
